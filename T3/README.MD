### Enunciado - Trabalho 3

<hr>
<br>

Desenvolver um programa que faça uso da orientação à objetos, incluindo construtores, métodos e suas sobrecargas, e principalmente **herança e operadores**.

O programa desenvolvido foi um gerenciador de fichas para bibliotecas universitárias / escolares. É possível fazer a devolução, empréstimo ou doação de livros. Após o usuário digitar seus dados, as fichas são criadas. Cada matrícula pode pegar até 2 livros emprestados de uma vez.
